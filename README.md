# 📊 Financial Loan Applications Dashboard

## 📌 Project Overview
This project presents an interactive Excel dashboard analyzing financial loan application data.  
The dashboard provides insights into loan performance, funding distribution, delinquency trends, borrower segmentation, and geographic distribution.

It is designed to help stakeholders monitor key KPIs and identify business trends effectively.

---

## 🎯 Business Objectives

- Analyze total loan applications
- Compare funded amount vs amount received
- Identify percentage of delinquent loans
- Evaluate average interest rate and DTI ratio
- Understand loan purpose trends
- Track monthly application growth
- Analyze state-wise loan distribution
